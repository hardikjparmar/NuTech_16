E-Contraption

If you think that machines always simplify tasks, you are mistaken! Because in a contraption you have to do the exact opposite Participants would be given a simple task which they would have to accomplish using a contraption model.

The Contraption model is to be made using electrical and electronic component (Other components also allowed). Round One would constitute sending an abstract of the model. Based on it participants would be chosen for the second round which would consist of making a working model.
